# 나만의 로또 만들기

## 진행 방식

1. 구입 금액을 입력 받는다. (1,000원 단위로 입력 받으며, 알맞지 않은 입력일 경우 IllegalArgumentException을 발생시킨다.)
2. 구입 금액에 맞는 로또를 발행한다. (ex. 8,000원을 입력 받으면 로또 8장 발행)
   1. 랜덤값 추출은 `pickUniqueNumbersInRange()`를 활용
   2. 로또 값은 최대 6개(이미 구현되어 있음)
3. 당첨 번호를 입력 받는다. (6개)
4. 보너스 번호를 입력 받는다. (1개)
5. 당첨 통계를 낸다. (수익률 계산)

## 기능 목록

1. 구입 금액을 입력 받는다.
   - `구매금액을 입력해 주세요.` 문구를 출력
   - Console.readLine() 활용하여 금액을 입력 받음
   - 1,000원 단위인지 체크(1,000원 단위가 아닐 경우 IllegalArgumentException)
2. 구입 금액에 맞는 로또를 발행한다.
   - `n개를 구매했습니다.` 문구를 출력
   - n개의 로또 번호를 발행(format = "[n1, n2, n3, n4, n5, n6]")
3. 당첨 번호를 입력 받는다.
   - `당첨 번호를 입력해 주세요.` 문구를 출력
   - Console.readLine() 활용하여 당첨 번호 6개를 입력 받음
   - 번호 개수가 6개가 아니거나, 중복된 숫자가 있다면 다시 입력 받음
4. 보너스 번호를 입력 받는다.
   - `보너스 번호를 입력해 주세요.` 문구를 출력
   - Console.readLine() 활용하여 보너스 번호 1개를 입력 받음
   - 번호 개수가 1개가 아니거나, 당첨 번호가 이미 포함된 번호라면 다시 입력 받음
5. 당첨 통계를 낸다. (수익률 = 당첨 금액 / 구입 금액, 소수점 둘째 자리에서 반올림)  
   ex)
   ```text
   당첨 통계
   ---
   3개 일치 (5,000원) - 1개
   4개 일치 (50,000원) - 0개
   5개 일치 (1,500,000원) - 0개
   5개 일치, 보너스 볼 일치 (30,000,000원) - 0개
   6개 일치 (2,000,000,000원) - 0개
   총 수익률은 62.5%입니다.
   ```
